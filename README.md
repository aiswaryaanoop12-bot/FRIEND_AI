# 🤖 FRIEND AI

An AI-powered chatbot built using FastAPI, Groq API, HTML, CSS, and JavaScript.

This project provides a modern chat interface where users can interact with an AI assistant and receive real-time responses.

---

## 🚀 Features

- AI-powered responses using Groq LLM
- FastAPI backend
- Modern chat interface
- User messages on the right
- AI messages on the left
- Enter key to send messages
- Copy AI response button
- Clear chat functionality
- Character counter
- Responsive UI
- Animated Galaxy Theme

---

## 🛠️ Tech Stack

### Backend
- Python
- FastAPI
- Groq API
- Pydantic
