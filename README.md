# 🤖 FRIEND AI

An AI-powered chatbot built using FastAPI, Groq API, HTML, CSS, and JavaScript.

This project provides a modern chat interface where users can interact with an AI assistant and receive real-time responses.

---

## 🚀 Features

- AI-powered responses using Groq LLM
- FastAPI backend
- Modern chat interface
- User messages on the right
- AI messages on the left
- Enter key to send messages
- Copy AI response button
- Clear chat functionality
- Character counter
- Responsive UI
- Animated Galaxy Theme

## ⚙️ Installation

### 1. Clone Repository

```bash
git clone https://github.com/aiswaryaanoop12/FRIEND-AI.git
```

### 2. Create Virtual Environment

```bash
python -m venv venv
```

### 3. Activate Environment

Windows:

```bash
venv\Scripts\activate
```

### 4. Install Dependencies

```bash
pip install fastapi uvicorn groq python-dotenv jinja2
```

### 5. Add API Key

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

### 6. Run Application

```bash
fastapi dev main.py
```

Open:

```text
http://127.0.0.1:8000
```
